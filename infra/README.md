# Infraestrutura (Savour)

Este diretório está reservado para arquivos de infraestrutura como código (IaC), configurações de deploy, pipelines e provisionamento.

## Estrutura futura
- `terraform/`: Arquivos Terraform para provisionamento de recursos de nuvem (AWS RDS, AWS ECS, VPC).
- `k8s/`: Manifestos Kubernetes para orquestração em ambientes de produção.
- `docker/`: Dockerfiles específicos ou scripts utilitários de container.
